# คู่มือการจัดการเนื้อหา (EED HALAL Website)

เนื่องจากเว็บไซต์ทำงานในรูปแบบ Static Hosting (ไม่มี Backend) การจัดการเนื้อหาต้องทำผ่านคอมพิวเตอร์ Local แล้วทำการอัปโหลด (Push) ขึ้นไปครับ

## 1. วิธีเพิ่ม/แก้ไข เมนูอาหาร (Food Menu)
ข้อมูลอาหารทั้งหมดอยู่ในไฟล์ `menu.json` โดยคุณสามารถใช้เครื่องมือช่วยจัดการได้ดังนี้:

1.  **เปิด Terminal** ในโฟลเดอร์โปรเจกต์นี้
2.  รันคำสั่ง: `npm run public:serve`
3.  เปิดไฟล์นี้ในเบราว์เซอร์: [http://localhost:5500/tools/menu-admin.html](http://localhost:5500/tools/menu-admin.html)
4.  ในหน้าเว็บให้กด **"โหลดจาก menu.json"** เพื่อเริ่มแก้ไข
5.  เมื่อแก้ไขเสร็จ กด **"Export menu.json"**
6.  นำไฟล์ที่โหลดได้มาวางทับไฟล์เดิมที่ชื่อ `menu.json` ในโฟลเดอร์หลัก

## 2. วิธีแก้ไขเมนูนำทาง (Header / Footer)
1.  เปิดไฟล์ [navigation.html](navigation.html) เพื่อแก้ลิงก์ด้านบน
2.  เปิดไฟล์ [footer.html](footer.html) เพื่อแก้ลิงก์ด้านล่าง

## 3. วิธีนำการเปลี่ยนแปลงขึ้นบนเว็บจริง
หลังจากแก้ไขไฟล์ในเครื่องเสร็จแล้ว (เช่น `menu.json` หรือ `navigation.html`) ให้ทำการส่งข้อมูลขึ้นโฮสต์ (Deploy):
*   หากใช้ Git: `git add .` -> `git commit -m "Update menu"` -> `git push`
*   หน้าเว็บจะใช้เวลาประมาณ 1-2 นาทีในการอัปเดตข้อมูลใหม่ครับ
